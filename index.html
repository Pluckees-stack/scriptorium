This file needs your Supabase keys before it can connect.

Open this file in a text editor and find the CONFIG block near the top of the script. In Supabase, go to Project Settings → API and copy two things: your Project URL, and the public key (labelled "anon public", or "publishable" on newer projects). Paste them in, save, and reopen. Never use the "service_role" / "secret" key here.
